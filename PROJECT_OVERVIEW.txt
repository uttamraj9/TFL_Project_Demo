PROJECT GOAL:
Transform raw Transport for London (TfL) passenger data into a production-ready
data warehouse with proper normalization, relationships, and analytical views.

  ✓ Proper primary keys on all tables
  ✓ Foreign key constraints for referential integrity
  ✓ Composite unique constraints to prevent duplicates
  ✓ 10 performance indexes on frequently queried columns
  ✓ Audit columns (created_at, updated_at)

DATA QUALITY:
  ✓ NOT NULL constraints on required fields
  ✓ UNIQUE constraints on natural keys

  ✓ Proper data types for each column

                          ANALYTICAL VIEWS
═══════════════════════════════════════════════════════════════════════════════

1. vw_station_summary
   - Complete station details with aggregated line information
   - Shows number of lines served and their names
   - Includes all service type flags

2. vw_annual_passenger_stats
   - Annual passenger statistics by station and line
   - Aggregated total, average, max, min passengers
   - Time-series ready

3. vw_busiest_stations
   - Stations ranked by passenger volume per year
   - Year-over-year ranking
   - Top N analysis ready

4. vw_line_stats
   - Line-level statistics
   - Station counts, interchange counts
   - Night tube coverage
